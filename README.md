# SentinelScan

Professional Security Analyzer written in Python.

## Features

- Security Header Scanner
- TLS/SSL Analysis
- Security Score
- TXT Reports
- JSON Reports
- HTML Reports
- CSV Reports
- Batch URL Scanning

## Installation

git clone https://github.com/DrAKULA12-lab/SentinelScan.git

cd SentinelScan

pip install -r requirements.txt

python src/main.py
